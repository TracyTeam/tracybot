1. RACE CONDITION IN MESSAGE HANDLING
-------------------------------------
Location: src/index.ts:144-187

Problem:
- message.part.updated (line 144) updates text incrementally as it streams
- message.updated (line 181) checks 'finish' property and creates tasklet immediately
- Tasklet is created before all part.updated events complete, resulting in incomplete text

Solution:
Wait for final complete message before creating tasklet. Remove 'finish' check from
message.updated handler and instead create tasklet only when message.part.updated
signals completion via finishReason:

```typescript
if (event.type === "message.part.updated" && event.properties?.part) {
    const part = event.properties.part
    if (part.type === "text" && part.text) {
        const msgId = part.messageID
        const msg = messages.get(msgId)
        if (msg) {
            msg.text = part.text
        }
    }
    
    // After part update, check if message is now complete
    const isComplete = (event.properties.part as any).finishReason === "stop"
    if (isComplete) {
        const msg = messages.get(msgId)
        if (msg && msg.agent === "build") {
            createTasklet(currentSessionId, msg)
        }
    }
}
```

Remove the tasklet creation logic from message.updated entirely.

interface MessageUpdatedEvent {
    type: "message.updated"
    properties: {
        info: {
            id: string
            role: string
            agent?: string
            parentID?: string
            finish?: boolean
        }
    }
}

interface MessagePartUpdatedEvent {
    type: "message.part.updated"
    properties: {
        part: {
            messageID: string
            type: "text"
            text: string
            finishReason?: string
        }
    }
}

interface SessionCreatedEvent {
    type: "session.created"
    properties: {
        info: { id: string }
    }
}

type PluginEvent = SessionCreatedEvent | MessageUpdatedEvent | MessagePartUpdatedEvent

C) Validate tool args with path bounds checking:

```typescript
"tool.execute.before": async (input, output) => {
    if (!EDIT_TOOLS.includes(input.tool)) return

    if (!input.args || typeof input.args !== "object") {
        await client.app.log({
            body: {
                service: PLUGIN_NAME,
                level: "error",
                message: `Invalid tool args`,
                extra: { input },
            },
        })
        return
    }

    const filePath = input.args.filePath
    if (!filePath || typeof filePath !== "string") {
        await client.app.log({
            body: {
                service: PLUGIN_NAME,
                level: "error",
                message: `Missing or invalid filePath`,
            },
        })
        return
    }

    // Validate path is within repo (prevent path escape)
    const resolvedPath = path.resolve(repoRoot, filePath)
    if (!resolvedPath.startsWith(repoRoot)) {
        await client.app.log({
            body: {
                service: PLUGIN_NAME,
                level: "error",
                message: `Path escape attempt detected: ${filePath}`,
            },
        })
        return
    }
}
```
